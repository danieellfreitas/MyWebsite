# Documentação do Projeto — MeuSite

Bem-vindo à documentação oficial do **MeuSite**. Este repositório de documentos serve como a única fonte de verdade para a arquitetura visual, padrões de desenvolvimento, rotinas de deploy e diretrizes de otimização para motores de busca (SEO) do projeto.

O **MeuSite** é um site estático profissional estruturado sob a tendência estética *Bento Grid*, operando estritamente em *Dark Theme* e focado em performance máxima através de tecnologias nativas.

---

## 📚 Índice de Conteúdos

### 1. Arquitetura do Sistema
*   [Visão Geral da Arquitetura (Overview)](arquitetura/overview.md): Descrição das tecnologias utilizadas (HTML5, CSS3 modular e JS nativo) e a organização do diretório de estilos.
*   [Sistema de Design (Design System)](arquitetura/design-system.md): Especificações exatas sobre paleta de cores (Zinc & Violeta), tipografia (Poppins), espaçamentos e regras visuais do Bento Grid.

### 2. Guias de Desenvolvimento
*   [Setup Local](guias/setup-local.md): Passo a passo simples para clonar e rodar o projeto localmente com recarregamento em tempo real a partir da pasta `/src`.
*   [Deploy & CI/CD](guias/deploy.md): Explicação completa sobre o fluxo automatizado de integração contínua (GitHub Actions) e publicação na Hostinger via FTP.
*   [SEO & Performance](guias/seo-performance.md): Boas práticas recomendadas para garantir excelente indexação orgânica em motores de busca e carregamento instantâneo de recursos.

> [!NOTE]
> Este projeto foi desenvolvido seguindo rígidos padrões de boas práticas de engenharia de software frontend: separação total entre conteúdo (HTML), estilo (CSS) e lógica (JS), nomenclatura de código estritamente em inglês e documentação em português do Brasil.
